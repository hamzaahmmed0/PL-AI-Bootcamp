# PureLogics AI/ML Trainee Program — Labs & Projects

Repository documenting my work throughout the **PureLogics "From Zero to AI Hero"** trainee program — a 12-week, onsite, intensive track covering Python foundations, machine learning, deep learning, and large language models.

## About the Program

| | |
|---|---|
| **Organization** | PureLogics |
| **Duration** | 3 months (12 weeks) |
| **Format** | Onsite, in-person training |
| **Schedule** | Evenings, 5:00 PM – 9:00 PM |
| **Focus** | Python, Machine Learning, Deep Learning, LLMs & AI Agents |
| **Extras** | Weekly soft skills sessions, mentorship from industry experts |

## Curriculum

### Month 1 — Foundations
- **Week 1:** Python & AI Essentials
- **Week 2:** Data Handling & Project Setup
- **Week 3:** Data Preprocessing & Exploratory Data Analysis
- **Week 4:** Machine Learning Algorithms in Action

### Month 2 — Deep Learning
- **Week 5:** Model Tuning & Deployment
- **Week 6:** Neural Networks & Deep Learning Fundamentals
- **Week 7:** CNNs & Computer Vision
- **Week 8:** Transformers & the Architecture Behind LLMs

### Month 3 — Applied AI
- **Week 9:** LLM Applications & RAG Systems
- **Week 10:** Generative Models & Prompt Engineering
- **Week 11:** AI Agents
- **Week 12:** Capstone Project & Special Topics
